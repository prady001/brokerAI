# Agente de Renovação — Pós-MVP

Este diretório está reservado para o Agente de Renovação, previsto para a **fase 2** do projeto.

## Escopo (pós-MVP)

- Monitoramento diário de apólices próximas do vencimento
- Régua de follow-up: 30d → 15d → 7d → 2d antes do vencimento
- Envio de propostas de renovação via WhatsApp
- Handoff para corretor humano após aceite ou casos especiais

## Dependência

Requer acesso à carteira de apólices via integração com o Agger (pós-MVP).
No MVP, a carteira é importada manualmente via CSV.
