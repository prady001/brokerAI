ANÁLISE PÓS-LEVANTAMENTO
________________________________________________________________________________

Após aplicar o roteiro, avalie:

  SINAL VERDE — projeto pode avançar no prazo previsto:
    [ ] Base de apólices exportável em até 1 semana
    [ ] Há pelo menos 1 pessoa técnica de referência na corretora
    [ ] Clientes já usam WhatsApp para contato com a corretora
    [ ] Existem regras claras sobre o que pode ser automatizado

  SINAL AMARELO — prazo pode precisar de ajuste:
    [ ] Base de apólices está parcialmente organizada
    [ ] Não há API no sistema atual (precisará de integração manual)
    [ ] Clientes são mistos em preferência de canal

  SINAL VERMELHO — risco alto, endereçar antes de iniciar:
    [ ] Dados de apólices estão em PDFs ou papéis físicos
    [ ] Não há clareza sobre quem aprova o quê internamente
    [ ] Carteira majoritariamente sênior ou PJ com exigência de humano
    [ ] Restrição regulatória não mapeada com seguradoras

________________________________________________________________________________

  Aplicar este roteiro antes do kickoff do M1 garante que o projeto
  comece com as informações certas e sem surpresas no meio do caminho.
